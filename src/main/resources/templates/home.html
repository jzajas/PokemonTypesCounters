Simple Counters Detailed Counters
This project is made of 2 sites. Each one allows user to choose up to 2 types of an enemy pokemon. Simple which returns what attack types are good or not and what types to defend with or avoid. Detailed one shows what attacks are good, ineffective and have no effect at all and shows how enemy's types interact with other types. Please note that some pokemon might have attacks with types other than their types and this application doesn't take that into consideration.
